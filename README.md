# Robotic Arm Kinematics & Path Planning

## Objective
To calculate robotic arm joint angles using inverse kinematics.

## Tools Used
- Python
- Google Colab

## Features
- Inverse Kinematics
- Joint Angle Calculation
- Robotic Arm Positioning

## Output
The program calculates the required joint angles for a robotic arm to reach a target coordinate.
